# MCDR_HibernateR_Ex

[插件原地址](https://github.com/HIM049/MCDR_HibernateR/)

相比于原先的版本，此版本新增了几个特性：

可以通过命令手动开关计时器

可以通过命令手动启停伪装服务器

黑名单增加正则表达式匹配支持

通过计时器循环定时检测，而不是通过判断玩家进出事件，防止因为mcdr的bug导致carpet假人小等于3名字时不触发

去掉了没意义的报错重试和关闭伪装服务器的长时间等待

修改了配置文件名防止配置与原先冲突，配置使用秒而不是分钟以更精确的进行检测

修复了BUG：

utf8配置文件乱码问题

统一配置文件访问，防止配置文件不同步
